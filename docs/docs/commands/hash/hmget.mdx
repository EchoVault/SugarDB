# HMGET

### Syntax
```
HMGET key field [field ...]
```

### Module
<span className="acl-category">hash</span>

### Categories
<span className="acl-category">fast</span>
<span className="acl-category">hash</span>
<span className="acl-category">read</span>

### Description
Retrieves the value of each of the listed fields from the hash.
