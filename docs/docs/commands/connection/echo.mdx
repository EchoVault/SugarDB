# ECHO

### Syntax
```
ECHO [message]
```

### Module
<span className="acl-category">connection</span>

### Categories
<span className="acl-category">connection</span>
<span className="acl-category">fast</span>

### Description
Sends a message to the SugarDB server and it returns the same message back.
